Emacs cheatsheet

1 Command multipliers

C-- M-- negative-argument
C-[0-9]+ M-[0-9]+ digit-argument
C-u universal-argument

2 Moving around

C-f forward-char
C-b backward-char
M-f forward-word
M-b backward-word
C-n next-line
C-p previous-line
C-a move-beginning-of-line
C-e move-end-of-line
M-e forward-sentence
M-a backward-sentence
M-< beginning-of-buffer
M-> end-of-buffer
M-g c goto-char

3 Killing and yanking

3.1 Killing

C-d delete-char
Del delete-backward-char
M-d kill-word
M-del backward-kill-word
C-k kill-line
M-k kill-sentence
C-x Del backward-kill-sentence
C-w kill-region
  kill-paragraph
  backward-kill-paragraph

3.2 Saving to kill-ring

M-w kill-ring-save

3.3 Yanking

C-y yank
M-y yank-pop

4 Marks and regions

C-space C-@ set-mark-command
C-x C-x exchange-point-and-mark
M-h mark-paragraph
C-x C-p mark-page
C-x h mark-whole-buffer

5 Editing goodies

C-t transpose-chars
M-t transpose-words
C-x C-t transpose-lines
  transpose-sentences
  transpose-paragraphs
M-c capitalize-word
M-u upcase-word
M-l downcase-word

6 Searching

C-s isearch-forward
C-r isearch-backward
C-s C-w isearch-yank-word
C-s C-y isearch-yank-line
C-s M-y isearch-yank-kill
C-s C-s isearch-repeat-forward
C-s C-r isearch-repeat-backward
C-M-s isearch-forward-regexp
C-M-r isearch-backward-regexp
M-s w isearch-forward-word

6.1 At a search prompt

Enter isearch-exit

6.2 Swiper

M-q swiper-query-replace
C-o hydra-ivy/body

7 Replacing

M-% query-replace
C-M-% query-replace-regexp

7.1 At replace prompt

C-r Enter recursive edit
C-w Delete match and recursive edit
^ Go back to previous match
E Edit replacement string

8 Macros

C-x ( <f3> kmacro-start-macro
C-x ) <f4> kmacro-end-macro
C-x e kmacro-end-and-call-macro
C-x C-k r apply-macro-to-region-line
  • To loop a macro until error or end of buffer, call it with 0 as a numeric argument.

9 Debugging

9.1 CPU/RAM Usage

  profiler-start
  profiler-report
